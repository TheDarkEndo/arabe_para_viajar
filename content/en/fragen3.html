How to deal with problems

How can I print the list?

    Start by clicking on any part of text within the index card frame with your left mouse button to ensure that the computer will print the correct part of the page. You should now print the page by clicking on the printer icon of the browser task bar. Alternatively you can choose the print command from the browser menu.

    You can also copy the content of the browser page into any word processing application (e.g. Word, OpenOffice etc.) where you can edit and print it.
    Just follow these steps:
    1. Use your mouse to select everything you would like to copy. (By pressing CTRL+A you will select the entire content of the window)
    2. Press CTRL+C to copy the content. Alternatively you may click your right mouse button and select COPY from the pop-up menu bar.
    3. Switch to your word processing application (e.g. Word, OpenOffice)
    4. Press CTRL+V to paste the selected parts into the word processing document. It is also possible to use the PASTE command in the taskbar.

My computer will not show some of the special characters!

    Here are three ways that may help you deal with this problem:

    a) The language course uses fonts that should be available on every computer, such as Verdana or Tahoma. One of these might have been removed from your computer. in that case you should re-instal it.

    b) Use the system settings to enable foreign language support. The missing characters should be available afterwards.

    c) If neither of these options have proved successful you may want to try a different browser (e.g. Firefox, Mozilla or Netscape instead of the Internet Explorer or Safari).

How do I enable non-latin alphabets (e.g., Cyrillic, Chinese, Arabic etc.)

    Windows:
    Choose the option "control panel" from your start menu.
    Proceed by clicking the header "regional and language options" followed by the option "languages" (top).
    You should now proceed by choosing the option "supplemental language support" and proceed to instal either the:
    - files for complex script and right-to-left languages
    - or the files for east-asian languages

    Mac OS X:
    Non-latin fonts should be pre-installed. If you cannot display the characters please go to your "system preferences" menu and select "international".
    You should proceed by selecting the language you want to learn.
    If you want to adapt your keyboard to be able to type in the foreign language you should make the necessary adjustments using the keyboard menu.

    Linux:
    In the main menu, choose "System", then successively "Configuration", "Packaging" and "Instal Software". Once you have opened the installation menu you should be able to list the available packages. Go to the letter "F" and instal any fonts you need.

    Additional possibility for all operating systems:
    If no complex alphabets are available in your computer you can easily instal them like this:

    Go to http://www.mozilla.org/support/firefox/options#fontsandcolors You will find instruction on how to instal support for additional languages. We recommend you select a unicode format.

Studying with the long-term memory option will not work properly.

    Please enable your browser to accept cookies.

    Using the Internet Explorer:

    • Choose "Tools" from your task bar.
    • Click the header "Internet options".
    • Choose "security" or "advanced" and the "privacy" (privacy - accept all cookies).
    • Choose "accept all cookies" (or choose the lowest security level).
    • Confirm by using the "ok" button.

    Using Firefox, Netscape oder Mozilla:

    • Choose "View" from the menu.
    • Select the header "Internet Options".
    • Click "Advanced".
    • Choose "Security/Cookies"
    • Select "Enable all Cookies" from the menu
    • Confirm by using the "ok" button.

    Using Safari:

    • Go to the "Safari" menu.
    • Select "Enable all cookies" from the security settings menu.
    • Confirm by using the "ok" button.

    If you are still experiencing problems with the Internet Explorer please switch the security settings of the browser to the lowest level. Alternatively you may want to try a different browser.

What browser settings should I choose?

    • Please enable JavaScript
    • Please enable cookies.
    • If you are working with the Internet Explorer you should switch the security settings to the lowest level.

    You should be able to make the appropriate selections in your browser's "settings" or "extra" menu.

The lessons will not show!

    Please enable JavaScript. If you are still having trouble you might want to try a different browser.

Some windows will not open!

    Please disable all pop-up blockers.

Do I have to get online?

    No, you do noot need to be online to work with the language course. If your browser tries to connect to the internet you should choose "work offline" from the menu bar.
