Fragen zum Start
Wie starte ich?

    Bitte starten Sie einfach die Datei, die auf Ihrer Verpackung aufgedruckt ist, mit einem beliebigen Internet-Browser (z.B. Internet-Explorer, Netscape, Mozilla, Firefox, Safari...)

Welche Einstellungen sind im Browser nötig?

    • Bitte aktivieren Sie JavaScript
    • Klicken Sie auf "Cookies aktivieren"

    Die Einstellungen können Sie in Ihrem Browser vornehmen unter:
    Internet Explorer: Extras/Internetoptionen
    Firefox/Mozilla/Netscape: Extras/Einstellungen
    Safari: Safari/Einstellungen

Wie fange ich am besten an?

    Lernen Sie am besten zuerst am Bildschirm die Vokabeln und hören Sie sich diese einige Male an (Auswahlmenü: Vokabeln / Vokabelliste). Drucken Sie anschließend die Vokabeln als Karteikarten aus, wobei Sie Vorder- und Rückseite getrennt bedrucken sollten (Auswahlmenü: Vokabeln / Karteikarten, klicken Sie danach rechts oben auf das Druckersymbol). Lernen Sie die Karteikarten und testen Sie anschließend Ihre Kenntnisse am Computer (Auswahlmenü: Vokabeln / Test).

Es kommen keine Lektionen!

    Bitte aktivieren Sie JavaScript oder verwenden Sie einen anderen Internet-Browser.

Muss ich online sein?

    Nein, Sie müssen nicht online sein, um den Kurs benutzen zu können. Wenn Ihr Browser trotzdem versucht, eine Internetverbindung aufzubauen, so klicken Sie in der oberen Menüleiste Ihres Browsers auf "Offline arbeiten".
