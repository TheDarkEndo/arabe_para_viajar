Singhalesische Schrift

Bevor Sie mit dem Kurs arbeiten können, müssen Sie Ihren Computer erst einmal für Singhalesisch fit machen, damit er die besonderen Schriftzeichen des Singhalesischen auch lesen kann.
Hierzu müssen Sie sich aus dem Internet einen sogenannten Font herunterladen. Was Sie hierfür tun müssen, werden wir Ihnen im folgenden genau erläutern.
  1. Starten Sie Ihren Computer und verbinden Sie sich mit dem Internet.
  2. Starten sie eine Suchmaschinenseite: z.B. GOOGLE, YAHOO, ALTAVISTA ...
  3. Geben Sie in die Suchmaske folgende Worte ein: Madhura Font.
  4. Laden Sie jetzt den Font madhura.ttf von einer der angezeigten Webseiten herunter.
  5. Dieser Font ist ein Truetype-Font, der alle singhalesischen Schriftzeichen besitzt.
  6. Wenn Sie den Font herunterladen haben, installieren Sie diesen auf Ihrem Computer. Für Windowsbenutzer lesen Sie einfach weiter. Für Linuxuser, sehen Sie sich bitte Ihr Handbuch für Schrifteninstallation unter Linux an.
  7. Schrifteninstallation unter Windows: Wenn es sich um eine .zip Datei handelt, müssen Sie die Datei noch entpacken. Wenn es sich um eine .exe Datei handelt, doppelklicken Sie auf die Datei und sie entpackt sich von alleine.
  8. Wenn Sie die Datei als madhura.ttf vorliegen haben, kopieren Sie diese bitte in folgenden Ordner: C:/WINDOWS/Fonts
  9. Jetzt ist der Font installiert. Bevor Sie den Font benutzen können, müssen Sie nochmal den Computer neu starten.
Wenn Sie nun den Kurs starten, werden Sie alle singhalesischen Schriftzeichen sehen können.
