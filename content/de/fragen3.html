Fehlersuche / Problemlösungen

Wie kann die Liste ausgedruckt werden?

    Bitte klicken Sie zuerst mit der linken Maustaste auf den Text und danach auf den Druckbutton Ihres Browsers (oder wählen Sie im Menü Ihres Browsers 'Drucken' bzw 'Print' aus).

    Sie können außerdem den Fensterinhalt kopieren, in jedes beliebige Textverarbeitungsprogramm (z.B. Word, OpenOffice,...) einfügen, nachbearbeiten und dort ausdrucken.
    Das funktioniert folgendermaßen:
    1. Markieren Sie alles, was Sie kopieren möchten (mit der Tastenkombination STRG+a markieren Sie das ganze Fenster)
    2. Drücken Sie entweder STRG+c oder die rechte Maustaste (klicken Sie danach auf KOPIEREN)
    3. Wechseln Sie in Ihr Textverarbeitungsprogramm (z.B. Word, OpenOffice,...)
    4. Drücken Sie STRG+v oder klicken Sie auf EINFÜGEN

Einige Sonderzeichen funktionieren nicht!

    Hierzu gibt es drei Lösungsmöglichkeiten:

    a) Der Sprachkurs verwendet Schriftarten, die auf jedem Computer vorhanden sind, z.B. Verdana oder Tahoma. Wenn eine dieser Schriften auf Ihrem Computer entfernt wurde, so installieren Sie diese bitte wieder.

    b) Unter der Systemsteuerung können Sie die Fremdsprachenunterstützung aktivieren. Danach werden alle Sonderzeichen verfügbar.

    c) Falls auch das nicht funktioniert, so können Sie auch einen anderen Browser (z.B. Firefox, Mozilla oder Netscape statt Internet Explorer oder Safari) verwenden.

Wie installiere ich nicht-lateinische Schriften (Kyrillisch, Chinesisch, Arabisch,...)?

    Windows:
    Klicken Sie im Startmenü auf "Systemsteuerung",
    dort auf "Regions- und Sprachoptionen",
    danach auf "Sprachen" (oben),
    danach auf "Zusätzliche Sprachunterstützung",
    aktivieren Sie:
    - Dateien für Sprachen mit komplexen Schriftzeichen
    - Dateien für ostasiatische Sprachen

    Mac OS X:
    Nicht-lateinische Schriften sind bereits vorinstalliert. Falls sie nicht dargestellt werden, so klicken Sie auf "Systemeinstellungen", dort auf "Landeseinstellungen" und dort auf "Sprachen".
    Wählen Sie aus der Liste Ihre gewünschten Sprachen aus.
    Wenn Sie mit einer anderen Tastatur tippen möchten, die ein anderes Tastaturschema hat, so können Sie diese unter "Tastaturmenü" auswählen.

    Linux:
    Klicken Sie im "Kontrollzentrum" auf "Software installieren" und danach auf "Grafische Arbeitsfläche", "KDE-Arbeitsplatz" oder "GNOME-Arbeitsplatz" und aktivieren Sie dort Ihre gewünschten Fonts bzw. den gewünschten Sprachsupport für KDE.

    Weitere Möglichkeit für alle Betriebssysteme:
    Falls auf Ihrem Computer keine komplexen Schriften verfügbar sind, so können Sie diese leicht nachinstallieren.

    Unter http://www.firefox-browser.de/wiki/FAQ:Fehlende_Schriften finden Sie hierzu detaillierte Anleitungen sowie weitere Schriften, so dass Sie in Zukunft alle komplexen Schriften (Chinesisch, Hindi, Arabisch,...) darstellen können. Wir empfehlen Ihnen, eine Unicode-Schrift zu installieren.

Das Langzeitgedächtnis-Lernen funktioniert nicht richtig!

    Bitte aktivieren Sie Cookies in Ihrem Browser.

    Unter Internet Explorer:

    • Gehen Sie ins Menü "Extras"
    • Klicken Sie auf "Internetoptionen"
    • Klicken Sie auf "Datenschutz" oder "Erweitert"
    • Klicken Sie auf "Alle Cookies annehmen" (oder schieben Sie den Regler nach unten)
    • Klicken Sie auf OK

    Unter Firefox, Netscape oder Mozilla:

    • Gehen Sie ins Menü "Ansicht" oder "View"
    • Klicken Sie auf "Einstellungen" oder "Internet Options"
    • Klicken Sie auf "Erweitert" oder "Advanced"
    • Klicken Sie auf "Sicherheit/Cookies" oder "Security/Cookies"
    • Klicken Sie auf "Alle Cookies akzeptieren" oder "Enable all Cookies"
    • Klicken Sie auf OK

    Unter Safari:

    • Gehen Sie ins Menü "Safari"
    • Klicken Sie auf "Einstellungen"
    • Klicken Sie auf "Sicherheit"
    • Klicken Sie auf "Cookies immer akzeptieren"
    • Klicken Sie auf OK

Welche Einstellungen sind im Browser nötig?

    • Bitte aktivieren Sie JavaScript
    • Klicken Sie auf "Cookies aktivieren"

    Die Einstellungen können Sie in Ihrem Browser vornehmen unter:
    Internet Explorer: Extras/Internetoptionen
    Firefox/Mozilla/Netscape: Extras/Einstellungen
    Safari: Safari/Einstellungen

Es kommen keine Lektionen!

    Bitte aktivieren Sie JavaScript oder verwenden Sie einen anderen Internet-Browser.

Muss ich online sein?

    Nein, Sie müssen nicht online sein, um den Kurs benutzen zu können. Wenn Ihr Browser trotzdem versucht, eine Internetverbindung aufzubauen, so klicken Sie in der oberen Menüleiste Ihres Browsers auf "Offline arbeiten".

Funktioniert der Sprachkurs auch auf dem Pocket PC?

    Ja, viele Lernmethoden können Sie auch auf dem Pocket PC verwenden. Näheres dazu finden Sie in der Anleitung zum Pocket PC.
