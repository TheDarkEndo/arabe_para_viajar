Lernhilfen
Dieses Programm liefert Ihnen zahlreiche Lernhilfen, die Ihnen das Vokabelpauken erleichtern.

7er Wiederholung:

    Innerhalb der Tests erhalten Sie nach jeweils sieben nicht gewussten Wörtern eine Kurzwiederholung.
    Ein Fortschrittsbalken gibt Ihnen eine zeitliche Orientierung. Ist er ganz grün, so ist eine Minute vergangen.
    Erfahren Sie hier, warum eine 7er Wiederholung so effektiv ist.

Pause nach 30 Minuten:

    Innerhalb der Tests erscheint nach jeweils 30 Minuten ein Pausenzeichen. Hier können Sie eine fünfminütige Pause einlegen, bis die Ampel wieder grün wird:

    Erfahren Sie mehr über optimale Pausenlängen.

Zeithinweise beim Lernen einzelner Vokabeln:

    Damit Sie sich orientieren können, wie lange Sie sich bereits eine Vokabel einprägen, finden Sie unterhalb der Vokabel einen diskreten Zeithinweis: Alle 5 Sekunden erscheint ein neuer Punkt am Bildschirm.

Eingeblendete Zitate:

    An vielen Stellen im Programm erhalten Sie kurze eingeblendete Zitate mit weiteren Tipps, die Ihnen das Lernen erleichtern sollen.

Effektive Wiederholungen:

    In den Tests lernen Sie die Wörter solange, bis Sie sie mindstens einmal richtig eingegeben haben. Bereits gewusste Vokabeln müssen Sie nur 1x eingeben, lediglich die nicht gewussten Wörter werden wiederholt.
