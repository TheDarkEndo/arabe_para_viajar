How to work with the ultra long term memory test

 
The ultra long term memory test is based on the same principles as the standard long term memory test. There is, however, a significant difference:
Words that you got right will not be repeated the following day but only after a longer interval.

Level Interval Meaning
Level 5 No additional interval Words you did not get right will be repeated the following day.
Level 4 1 day interval On level 4 (=You got this word right first time before) words will not be repeated during the next session but only during the one after that.
Level 3 2 day interval On level 3 (=You got this word right twice in a row) you will only be asked the same word again three sessions later.
Level 2 4 day interval
Level 1 8 day interval On level 1 a word has almost made it into your long term memory. Usually, you will be able to remember these words for a very long time. This is why on the ultra long term memory test there will be an 8 day interval before the final repetition.

The scientific principle behind the ultra long term memory method:

The longer the time between two repetitions of a word the better you will keep it in mind. It is best if the interval between two repetitions is twice as long as the preceeding one. This is precisely how the ultra long term memory test works.

Recommendation:

If you have enough time on hand (several weeks) to learn the language you will achieve the best results using this method. It may take you longer to store a word in your long term memory (20 days on average taking into account the intervals as opposed to a week with the standard method) but you will notice that the vocabulary is rooted in your brain even more deeply.

If your aim is to learn the language as quickly as possible you had rather choose the standard long term memory method. Even so you should retake the long term memory test after three to six months.

Hinweis:

Using your code you can flexibly switch between the different long term memory methods.
