5. Medien


Benutzen Sie regelmäßig wolofe Medien, um Ihre Sprachkenntnisse zu verbessern und um Ihr Reiseland näher kennenzulernen. Deshalb erfahren Sie hier, unter anderem, wo Sie im Netz wolofe Zeitungen oder Radiosender finden.

Zeitungen

Wenn Sie möchten, können Sie senegalesische Zeitungen auch im Internet lesen. Leider gibt es noch keine senegalesische Tageszeitung, die im Internet auf Wolof schreibt, da Französisch immer noch Hauptsprache der Zeitungen ist. So können Sie zumindest Ihr Französisch auf eine interessante Art und Weise verbessern und gleichzeitig aktuelle Neuigkeiten aus dem Land erfahren. Wenn Ihnen Vokabeln fehlen, schlagen Sie diese einfach in einem gewöhnlichen Wörterbuch oder in einem deutsch-französischen Online-Wörterbuch nach.

Nachfolgend finden Sie eine Auswahl von senegalesischen Tageszeitungen in französischer Sprache mit ihrem Webauftritt:
Le Soleil: www.lesoleil.sn
Sud quotidien: www.sudonline.sn
Walfadjri: www.walf.sn
Le Quotidien: www.lequotidien.sn
L' Observateur: www.lobservateur.sn

Web-Radios aus Senegal

Um einen senegalesischen Radiosender hören zu können, müssen Sie mittlerweile nicht mehr im Land vor Ort sein. Im Internet kann man mit etwas Geduld über eine Suchmaschine verschiedene Online-Sender finden. Darunter finden sich sowohl Sender in Französisch als auch in Wolof.

Dies ist eine gute Gelegenheit, das Gelernte einem Test zu unterziehen und das Sprachverständnis der gesprochenen Sprache zu üben. Probieren Sie es doch einmal aus! Hier finden Sie eine erste kleine Auswahl:

Radio Walf Web (zugehöriger Sender zur Tageszeitung Walfadjri)
www.walf.sn/radio

Radio Sud FM (zugehöriger Sender zur Tageszeitung Sud Quotidien)
www.sudonline.sn

Radio RFM (Radio Futurs Médias) 94.0
www.futursmedias.net

Radio Nostalgie la Legende 90.3
www.nostalgie.sn

Senegal im Internet

Senegalesische Webseiten haben die Endung ".sn". Die meisten Seiten sind in französischer Sprache, aber man kann mit etwas Geduld über Suchmaschinen auch Webseiten in Wolof finden. Die offizielle Webseite der senegalesischen Regierung finden Sie auf folgender Seite: www.gouv.sn

Die offizielle Seite der Universität von Dakar (Université Cheikh Anta Diop) finden Sie unter folgender Adresse: www.ucad.sn

Ein interessante Seite zu Medien, Kommunikation, Reise und vielem mehr aus dem Senegal ist zum Beispiel: www.orange-info.sn

Aktuelle Informationen zu Politik, Sozialem, Sport etc. können Sie auch auf folgender Seite finden: www.homeviewsenegal.sn

Ansonsten finden Sie natürlich reichlich Information auf den Internetseiten der oben genannten Zeitungen bzw. Radiosender.

Telefonieren

Die internationale Vorwahl für Senegal ist 00221.

(Deutschland hat 0049, Österreich 0043 und die Schweiz 0041)

Falls Sie sich im Senegal in einer Notsituation befinden, bekommen Sie Hilfe, wenn Sie eine dieser Nummern wählen:

Polizei
17
Feuerwehr / Ambulanz
18
