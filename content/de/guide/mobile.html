Die mobile Version Ihres Sprachkurses für Pocket-PC und Smartphone


| Übersicht |
 

Windows Mobile

Sie können viele Funktionen dieses Sprachkurses auch auf Ihrem Pocket-PC verwenden. Wir haben hierzu eine spezielle Version für mobile Endgeräte entwickelt, die auch unter Windows Mobile ab 5.0 funktioniert – selbst bei einer Bildschirmbreite von nur 240 oder 320 Pixeln.

Kopieren Sie dazu einfach den kompletten Kurs auf Ihren Pocket-PC und starten Sie mobile.html mit dem Internet Explorer Mobile oder Opera Mini.

Wenn Sie eine Sprache mit komplexen Schriftzeichen (Chinesisch, Arabisch, Hindi, ...) lernen, so benötigen Sie unter Umständen noch einen Unicode-Font, um diese anzeigen zu können.

Kopieren Sie hierzu einen Unicode-Font (z. B. Arial Unicode) von Ihrem Computer ("C:/Windows/Fonts" oder "C:/Windows/Schriftarten") auf Ihren Pocket-PC (unter "Mein Gerät/Windows/Schriftarten").
 

Google Android

So funktioniert der Sprachkurs / das Wörterbuch unter Android:
 
1. Starten Sie den Browser
2. Tragen Sie in der Adressleiste die folgende Zeile ein:
 
Wenn Ihr Display groß genug ist:
file:///sdcard/[KURS]/index.html
ansonsten können Sie auch die mobile Version benutzen:
file:///sdcard/[KURS]/mobile.html
 
Statt [KURS] geben Sie einfach den Pfad zu Ihrem Kurs ein, also z.B. file:///sdcard/Franzoesisch_Expresskurs/index.html
file:///sdcard/Hindi_Basiskurs/index.html
file:///sdcard/Tuerkisch_Aufbaukurs/index.html
file:///sdcard/Aserbaidschanisch_Woerterbuch/index.html
 
Klicken Sie danach auf "GO!".
 
Legen Sie danach sofort ein neues Lesezeichen an!
 
Falls der Browser die Meldung "Die angeforderte Datei wurde nicht gefunden" ausgibt, so haben Sie sich entweder vertippt (z.B. Unterstrich mit Bindestrich verwechselt) oder Ihr Pfad lautet anders (z.B. wenn Sie keine SD-Card verwenden).
 
Öffnen Sie in diesem Fall eins der folgenden Programme:
  • OI File Manager
  • Glance
  • FileBrowser
und schauen Sie nach, wo Ihr Kurs gespeichert wurde.
Die meisten gängigen Telefone verwenden momentan (Micro-)SD-Karten, aber falls Sie eine andere Karte verwenden, so könnte der Pfad z.B.
file:///cfcard/Franzoesisch_Expresskurs/index.html
file:///compactflash/Franzoesisch_Expresskurs/index.html
oder ähnliches lauten. Dies können Sie mithilfe eines der oben angegebenen Programme feststellen.
Falls sich auf Ihrem Handy weder OI File Manager, noch Glance oder FileBrowser befinden, so klicken Sie auf "Market" und gehen Sie auf "Suche", um danach zu suchen.
 

iPhone

Leider funktioniert der Sprachkurs nicht auf dem iPhone, da momentan Dateien ausschließlich über den iTunes-Store auf das iPhone kopiert werden können.
