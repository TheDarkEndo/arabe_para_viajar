Häufig gestellte Fragen / Problemlösungen
Wie fange ich am besten an?

Lernen Sie am besten zuerst am Bildschirm die Vokabeln und hören Sie sich diese einige Male an (Auswahlmenü: Vokabeln / Liste). Drucken Sie anschließend die Vokabeln als Karteikarten aus, wobei Sie Vorder- und Rückseite getrennt bedrucken sollten (Auswahlmenü: Vokabeln / Karteikarten, Druckversionen). Lernen Sie die Karteikarten und testen Sie anschließend Ihre Kenntnisse am Computer (Auswahlmenü: Vokabeln / Test).

Probieren Sie beide aus; je nachdem, ob Sie z.B. Internet Explorer, Netscape oder Opera verwenden, wird entweder das erste oder das zweite besser funktionieren.
Alle Vokabeln und Texte wurden von Muttersprachlern aufgenommen und können von Ihnen mit einem Klick angehört werden. So lernen Sie eine klare, akzentfreie Aussprache.

Es kommt kein Ton!

Es gibt zwei verschiedene Bilder für die Sprachwiedergabe (einen Pfeil und einen Lautsprecher). Wenn einer der beiden nicht funktioniert, bitte verwenden Sie den anderen. Sie können auch einen anderen Internet-Browser verwenden.

Der Media Player verdeckt die Sicht auf die Texte!

Es gibt zwei verschiedene Bilder für die Sprachwiedergabe (einen Pfeil und einen Lautsprecher). Sie können probieren, welche der beiden Ausgabemöglichkeiten für Ihren Computer besser geeignet ist. Da der Sprachkurs betriebssystemunabhängig arbeitet, wurden zwei verschiedene Möglichkeiten programmiert.
Für den Fall, dass sich bei jeder Sprachausgabe ein Media Player öffnet und die Sicht verdeckt, haben Sie folgende Möglichkeiten:

1. Möglichkeit: Installieren Sie einen anderen Media Player. Unter Windows gibt es z.B. den Windows Media Player, Win Amp, Real Player sowie viele andere.

2. Möglichkeit: Verschieben Sie den Media Player an den Bildschirmrand, so dass er nicht mehr stört. Bei jeder weiteren Sprachausgabe stellt er sich zwar in den Vordergrund, versperrt dann aber nicht mehr die Sicht.

3. Möglichkeit: Verwenden Sie einen anderen Internet-Browser für den Sprachkurs. Jeder Internet-Browser gibt Töne anders aus. Es gibt momentan u.a. folgende Internet-Browser: Internet-Explorer, Netscape, Mozilla, Galeon,...

4. Möglichkeit: Klicken Sie auf das andere Bild! Es gibt zwei verschiedene Bilder für die Sprachwiedergabe (einen Pfeil und einen Lautsprecher). Wenn eine der beiden Möglichkeiten nicht so gut auf Ihrem Computer funktioniert, so verwenden Sie die zweite.

Es kommen keine Lektionen!

Bitte aktivieren Sie JavaScript oder verwenden Sie einen anderen Internet-Browser.

Einige Fenster öffnen sich nicht!

Bitte schalten Sie sämtliche PopUp-Blocker aus.

Muss ich online sein?

Nein, Sie müssen nicht online sein, um den Kurs benutzen zu können.

Wie starte ich?

Bitte starten Sie einfach die Datei, die auf Ihrer Verpackung aufgedruckt ist, mit einem beliebigen Internet-Browser (z.B. Internet-Explorer, Netscape, Mozilla, Galeon,...)

Wie kann die Liste ausgedruckt werden?

Bitte klicken Sie zuerst mit der linken Maustaste auf den Text und danach auf den Druckbutton Ihres Browsers (oder wählen Sie im Menü Ihres Browsers 'Drucken' bzw 'Print' aus).

Sie können außerdem den Fensterinhalt kopieren, in jedes beliebige Textverarbeitungsprogramm (z.B. Word, OpenOffice,...) einfügen, nachbearbeiten und dort ausdrucken.
Das funktioniert folgendermaßen:
1. Markieren Sie alles, was Sie kopieren möchten (mit der Tastenkombination STRG+a markieren Sie das ganze Fenster)
2. Drücken Sie entweder STRG+c oder die rechte Maustaste (klicken Sie danach auf KOPIEREN)
3. Wechseln Sie in Ihr Textverarbeitungsprogramm (z.B. Word, OpenOffice,...)
4. Drücken Sie STRG+v oder klicken Sie auf einfügen

Einige Sonderzeichen funktionieren nicht!

Hierzu gibt es drei Lösungsmöglichkeiten:

a) Der Sprachkurs verwendet Schriftarten, die auf jedem Computer vorhanden sind, z.B. Verdana oder Tahoma. Wenn eine dieser Schriften auf Ihrem Computer entfernt wurde, so installieren Sie diese bitte wieder.

b) Unter der Systemsteuerung können Sie die Fremdsprachenunterstützung aktivieren. Danach werden alle Sonderzeichen verfügbar.

c) Falls auch das nicht funktioniert, so können Sie auch einen anderen Browser (z.B. Netscape statt Internet Explorer) verwenden.

Das Langzeitgedächtnis-Lernen funktioniert nicht richtig!

Bitte aktivieren Sie Cookies in Ihrem Browser.

Unter Internet Explorer:

  • Gehen Sie ins Menü "Extras"
  • Klicken Sie auf "Internetoptionen"
  • Klicken Sie auf "Datenschutz" oder "Erweitert"
  • Klicken Sie auf "Alle Cookies annehmen" (oder schieben Sie den Regler nach unten)
  • Klicken Sie auf OK

    Unter Netscape oder Mozilla:

  • Gehen Sie ins Menü "Ansicht" oder "View"
  • Klicken Sie auf "Einstellungen" oder "Internet Options"
  • Klicken Sie auf "Erweitert" oder "Advanced"
  • Klicken Sie auf "Sicherheit/Cookies" oder "Security/Cookies"
  • Klicken Sie auf "Alle Cookies akzeptieren" oder "Enable all Cookies"
  • Klicken Sie auf OK

    Für den sehr seltenen Fall, dass Cookies nun immer noch nicht funktionieren sollten, ändern Sie unter "Einstellungen" die Sicherheitsstufe in "sehr niedrig". Statt Internet Explorer können Sie auch Netscape verwenden oder umgekehrt.
